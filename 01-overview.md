# Overview

WazzPay berperan sebagai **biller platform** yang membantu merchant membuat dan menerima pembayaran tagihan dari pelanggan mereka.

Merchant dapat membuat bill melalui:

- Dashboard B2B WazzPay
- Open API B2B

Setelah bill dibuat, sistem akan menghasilkan **payment link** yang dapat dikirim ke pelanggan.

Pelanggan dapat membayar menggunakan:

- QRIS
- Virtual Account
- DANA
- OVO

Setelah pembayaran berhasil, WazzPay akan mengirim **webhook callback** ke sistem merchant.
